# Vaccine Bot

A Discord bot used to alert users when local covid-19 vaccination appointments became available.

## Motivation

Created during the pandemic to automate the process of checking vaccine appointment availability.

## Tech/framework used

<b>Built using</b>

- Python
- Discord.py

Hosted on Heroku.
